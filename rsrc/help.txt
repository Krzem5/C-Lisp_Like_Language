Usage: lll [ <option1> <option2> <option3> ... ] <file1> <file2> <file3> ...
Options:
  -c -> Enable compilation only mode
  -e -> Expand file paths
  -f <file> -> Treat <file> as input file (Usefull for files starting with a
               dash ('-'))
  -G -> Do not generate the output files
  -h -> Print this message
  -I <directory> -> Add <directory> to file include path
  -L -> Disable compiler logo
  -m -> Merge imports
  -o <output_file> -> Set output file to <output_file>. If there is only one
                      <file>, the output is written to <output_file>. If there
                      are multiple <file>s, each <file> output is written to
                      <output_file_directory>/<base_file_name>[.lllc | .asm].
                      If no <output_file> is supplied, then each <file> output
                      is written to <file_without_extension>[.lllc | .asm]
  -O0 -> No optimization
  -O1 -> Remove padding
  -O2 -> Remove padding + Global optimization (default)
  -O3 -> Remove padding + Global optimization + Removal of debug data
  -p -> Print objects
  -R -> Disable program execution mode
  -v -> Enable verbose mode
